# Natural Language Processing (NLP)

Natural Language Processing (NLP) is a field of Artificial Intelligence that focuses on enabling computers to process, analyze, and understand human language. It combines concepts from machine learning, linguistics, and data processing to extract meaning and perform language-related tasks.

In this course, the focus began with **text representation and encoding techniques**, including **One-Hot Encoding**, **Label Encoding**, **Bag of Words (BoW)**, **TF-IDF**, **Word2Vec**, and **FastText**, which transform textual data into numerical representations suitable for machine learning models.

The course also covered **sequence modeling and deep learning approaches** for NLP, including **Recurrent Neural Networks (RNNs)**, **Long Short-Term Memory (LSTM)** networks, and **Gated Recurrent Units (GRU)** for learning sequential patterns in text data. In addition, **Transformers** were introduced as a modern architecture for capturing contextual relationships and improving performance across a wide range of language tasks.

This repository contains assignments and implementations that demonstrate these techniques and their applications in natural language processing.
